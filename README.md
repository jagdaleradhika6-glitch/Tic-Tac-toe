🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe web application built using frontend technologies. This project demonstrates core concepts of JavaScript, DOM manipulation, and user interaction while recreating the classic two-player game in a modern UI.

🚀 Live Demo :- https://jagdaleradhika6-glitch.github.io/Tic-Tac-toe/

📌 Features
🟢 Two-player gameplay (X and O)
🎯 Automatic win detection (rows, columns, diagonals)
🤝 Draw condition handling
🔄 Reset / Restart game functionality
🚫 Prevents overwriting filled cells
📱 Responsive and clean UI design

⚙️ How It Works
The game is played on a 3×3 grid.
Players take turns marking empty cells with X or O.
After each move, the system checks for a winner.
If all cells are filled without a winner, the game ends in a draw.
Users can restart the game anytime using the reset button.
